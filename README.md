# Deadlock Simulation

Simulates deadlock scenarios for FIFO and Round-Robin systems 

##### Limits on T (# of tasks) and R (# of resource types):
Maximum number of tasks: 10
Maximum number of resource types: 10

##### Compile/Run

```
javac *.java
java Main ./input/input-01.txt
```
